# Spinder: The Music Discovery App

## Description
Spinder is a music discovery app designed to help users find new music based on their preferences. Utilizing Spotify's recommendation API, Spinder suggests tracks, artists, and albums tailored to each user's taste.

## Features
- Music recommendations based on user preferences
- Playlist creation and management
- Social profiles and chatting

## Installation
Ensure you have Java and Maven installed. Clone the repository and navigate into the project directory. Run the application using Maven:
```bash
git clone https://github.com/justinsoberano/Spinder
cd Spinder
mvn spring-boot:run
